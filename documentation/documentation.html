Create your own site

You can easily create your own site using Sections.

Go to Theme folder and delete all the Demo HTML files and create a file called index.html and paste this code inside it: 

<!DOCTYPE html>
<html lang="en">

<head>

    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <meta name="description" content="Multipurpose One Page Bootstrap Theme">
    <meta name="keywords" content="html5, css3, mobile first, responsive, one page bootstrap theme" />
    <meta name="author" content="KingStudio.ro">

    <!-- Favicon -->
    <link rel="icon" href="images/favicon.png">
    <!-- Page Title -->
    <title>SimplyCity - Multipurpose One Page Theme</title>
    <!-- Bootstrap CSS -->
    <link href="css/bootstrap.min.css" rel="stylesheet">
    <!-- CSS -->
    <link href="css/style.css" rel="stylesheet">
    <link href="css/owl.carousel.css" rel="stylesheet">
    <link href="css/owl.theme.css" rel="stylesheet">
    <link href="css/animate.css" rel="stylesheet">
    <!-- Fonts -->
    <link href='https://fonts.googleapis.com/css?family=Lato:300,400,400italic,700' rel='stylesheet' type='text/css'>
    <link href="css/font-awesome.min.css" rel="stylesheet" type="text/css">
    <link rel="stylesheet" href="css/linear-icons.css">

Edit Colors and Styles

You can edit the Colros and Styles of the theme easily, using SASS. For SASS you may need to use SASS preprocessor application like Scout APP for example.

Start and setup the SASS preprocessor and go to Themes / scss and open style.scss with a code editor to edit the theme's styles. Open _variables.scss to edit the colors, fonts and font styles:

// Font and Styles

$primary-font: 'Lato', sans-serif;
$regular: 400;
$bold: 700;
$letter-spacing: 0.5px;

// Theme Colors

$default: #777;
$default-hover: #555;
$primary: #9c64b8;
$primary-hover: #8853a1;
$primary-transparent: rgba(155, 100, 184, 0.6);
$primary-transparent-overlay: rgba(155, 100, 184, 0.8);
$success: #87c056;
$success-hover: #73a449;
$info: #4da5e2;
$info-hover: #428fc5;
$warning: #fbd255;
$warning-hover: #f4c63d;
$danger: #ee4343;
$danger-hover: #d73737;
$title-color: #333;
$text-color: #555;
$bg-color: #eee;
$white: #fff;
$white-transparent: rgba(255, 255, 255, 0.98) !important;
$light-grey: #eee;
$light-grey-transparent: rgba(245, 245, 245, 0.8) !important;
$grey: #ccc;
$grey-transparent: rgba(200, 200, 200, 0.8) !important;
$dark-grey: #333;
$dark-grey-transparent: rgba(150, 150, 150, 0.8) !important;
$black: #000;

Open _media-screens.scss to edit the Media Queries, to change the way the theme looks on different screen sizes.

Open _mixins.scss to edit the Mixins.

Open _bootstrap-elements.scss to edit the bootstrap elements like jumbotron, buttons, tabs, labels, carousel, etc.

you can preview the changes you do here by going to Theme / pages and open elements.html file in your browser and all the changes you do in _bootstrap-elements.scss will be seen live by refreshing your browser while you have elements.html open in it.

How to use Google Maps

Since Google have changed their Terms about Google Maps, we have to use Google Maps API in order to display Google Maps on Our website, so if you would like to display the Google Map on your site you will need to add your Google Maps API at the bottom of your main .html ( maybe index.html ), so by default we have a demo API KEY there, so you just have to replace it, as shown below:

- in index.html ( demo ) file at the bottom ( in javascript / scripts section ) you will see this script: 

        <script src="https://maps.googleapis.com/maps/api/js?key=AIzaSyBA40uewXP25u1A4o9u8ueBimZZwIdNLkY"></script>

- in the above script replace the code after js?key= with your own Google Maps API KEY which you can get from here

Setup Contact Form

You can setup the contact form to send emails to your email address in php / form-process.php file edit this code from line 35:

          $EmailTo = "hello@example.com";

Replace the dummy email address hello@example.com with your own email address

In form-process.php you can edit the subject of the emails you will get by editing this code, from line 36:

          $Subject = "New Message Received";

Replace New Message Received from the above code with the subject you want to see when you receive an email from this contact form
